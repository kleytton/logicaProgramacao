Estatísticas da equipe

Queremos criar, recuperar e adicionar informações sobre seu time favorito. Basquete, futebol, tênis ou pólo aquático, você escolhe. É seu trabalho criar dados usando as estruturas de dados JavaScript à sua disposição: arrays e objetos. Depois de criarmos essas estruturas de dados neste projeto, sinta-se à vontade para desafiar a si mesmo e obter insights delas. Por exemplo, você pode querer obter o número total de jogos que seu time jogou ou a pontuação média de todos os jogos.

    Populando Dados

  1. Vamos fazer uma estrutura de dados para armazenar as informações sobre nossa equipe. Declare uma variável const chamada team e defina-a como um objeto vazio.
  2. Nosso time tem jogadores e os jogos que eles jogaram. Vamos representar ambos adicionando duas propriedades ao seu objeto de equipe. Adicione uma propriedade _players e uma propriedade _games e inicialize ambas em arrays vazios.
  3. Em seguida, preencha o array empty_players com três jogadores. Cada jogador deve ser um objeto contendo três propriedades: firstName, lastName e age. Coloque cada jogador em uma nova linha para evitar que a linha fique muito longa.
  4. Vamos fazer o mesmo para nosso array _games. Preencha o array vazio com três jogos. Cada jogo deve ser um objeto contendo três propriedades: opponent, teamPoints, opponentPoints.

    5. Obtendo Dados

  5. Crie um método getter chamado players para recuperar a propriedade _players. Dentro do método getter, retorne a propriedade _players. Para o escopo deste projeto, não precisaremos criar métodos setter, pois não queremos que ninguém altere os dados salvos nas propriedades.
  6. Crie outro método getter chamado games para recuperar a propriedade _games. Dentro do método getter, retorne a propriedade _games.

    7. Adicionando Dados

  7. Queremos adicionar um novo jogador ao nosso time. Adicione um método .addPlayer() ao objeto de equipe. Esse método deve receber três parâmetros: newFirstName, newLastName e newAge. Dentro do método, crie um objeto player definindo os três parâmetros para serem os valores das três propriedades do objeto: firstName, lastName, age. Por fim, adicione o objeto ao array _players do time.
  8. Abaixo do objeto de equipe, vamos experimentar nosso novo método .addPlayer() para adicionar um novo jogador: Pernalonga, 76 anos. Registre a propriedade _players do time para verificar se nosso novo jogador foi adicionado.
  9. Nosso estatístico tem algumas informações novas para nós! Crie um método para adicionar os resultados do jogo chamado addGame que usa três parâmetros: newOpponent, newTeamPoints, newOpponentPoints. Dentro do método .addGame(), crie um objeto de jogo definindo os três parâmetros para serem os valores das três propriedades do objeto: opponent, teamPoints, opponentPoints. Adicione o objeto do jogo ao array _games do time.
  10. Finalmente, abaixo do nosso objeto de equipe, use o método .addGame() para adicionar um registro de um novo jogo. A nossa equipe jogou contra os 'Titans' onde somamos 100 pontos e o adversário marcou 98 pontos. Registre a propriedade _games do time para verificar se nosso novo registro de jogo foi adicionado corretamente.
