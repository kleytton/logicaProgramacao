    Início

  1. A variável story contém o parágrafo que iremos editar. Para editar a história, nós a transformamos em um array na linha 3. O método .split() separa a string história pelo caractere de espaço (' ') e armazena cada palavra como um elemento do array. Para ver o array com o qual trabalharemos ao longo da lição, registre storyWords no console. Depois de visualizar o array storyWords, comente a instrução console.log() antes de passar para a próxima tarefa.
  2. Para uma melhor comparação visual das histórias originais e editadas, queremos visualizar o array storyWords editado como uma string. Para alterar o array storyWords de volta para uma string legível, podemos invocar o método .join() em storyWords. Forneça ao método .join() um argumento de um caractere de espaço vazio (' ') para separar cada elemento do array com um espaço na string. Coloque a invocação do método .join() como um argumento de uma instrução console.log() para registrar a história final no console.

    3. Contando Palavras

  3. Agora é hora de começar a editar a história manipulando o array storyWords. Queremos poder ver as alterações, portanto, certifique-se de que seu console.log() da história unida seja a última linha de código em seu editor. Primeiro, acima da instrução console.log() que usa o método .join(), crie uma variável chamada count que armazena o número 0. Diretamente abaixo de count, use um método .forEach() para iterar sobre o array storyWords. Como argumento do método forEach(), crie uma função vazia para ser usada como função de retorno de chamada(callback function). Embora a sintaxe de seta, arrow function do ES6 seja recomendada para a função de retorno de chamada, sinta-se à vontade para usar qualquer sintaxe com a qual se sinta confortável.
  4. Para cada palavra no array storyWords, queremos que a variável de contagem seja incrementada em um. Adicione um parâmetro chamado word à função de retorno de chamada do método .forEach() a ser usado para armazenar o elemento atual ao iterar sobre o array storyWords. Cada vez que storyWord itera, incrementa a contagem em um. Abaixo do método .forEach(), registre a contagem para ver quantas palavras existem na história.

    5. Filtrando Palavras

  5. Uma contagem de palavras de 181 é um pouco longa para esta história. Vamos filtrar todas as instâncias da palavra "literally" para encurtar a história e remover a palavra desnecessária. Reatribuiremos a história filtrada à mesma variável storyWords aplicando o método .filter()! Ao longo do projeto, usaremos essa abordagem de reatribuir a variável storyWords para cada revisão da história. Abaixo de onde você registrou a variável count, reatribua a variável storyWords para igualar a invocação do método .filter() no array storyWords. Forneça ao método .filter() uma função de retorno de chamada com um parâmetro word.
  6. Abaixo da declaração da variável storyWords, observe a variável unnecessaryWord na linha 4. Queremos filtrar o valor de unnecessaryWord da história. Dentro do corpo da função de retorno de chamada do método filter(), retorne word somente se ela NÃO for igual a unnecessaryWord. Verifique a string history no console para garantir que não inclua a palavra "literally". A primeira instância de "literally" estava anteriormente na primeira frase.

    7. Substituindo palavras

  7. Agora que removemos as palavras desnecessárias, vamos cuidar de quaisquer palavras incorretas na história. Podemos reatribuir storyWords a um novo array de palavras com verificação ortográfica usando o método .map()! Reatribua storyWords para igualar a invocação do método .map() no array storyWords. Defina a palavra como um parâmetro da função callback de .map(). No corpo do callback, defina uma instrução condicional para verificar se o argumento da palavra é igual à variável mispelledWord. Se for, retorne a grafia correta da string, “beautiful” que é "beautifull". Se não, retorne word. Dê uma olhada na string story no console para ver as grafias corretas! Você pode ver uma ocorrência da palavra com verificação ortográfica na primeira frase da história.
  8. Uh oh, sua avó vai ler a história e tem uma palavra “ruim” nela! Vamos aplicar o método .findIndex() a storyWords para encontrar o índice da palavra inválida. Comece declarando uma variável chamada badWordIndex e definindo-a para a invocação de .findIndex() no array storyWords. A função callback .findIndex() deve verificar cada palavra para ver se ela é igual à variável badWord declarada na linha 6 e retornar o índice da palavra encontrada. Em seguida, registre badWordIndex no console.
  9. Agora que temos o índice da palavra ruim, podemos substituí-lo facilmente por algo mais apropriado. Acesse o elemento dentro do array storyWords que possui o índice de badWordIndex usando a notação de colchetes. Defina o elemento acessado igual à string mais apropriada, 'really'. Salve o código e verifique se o palavrão foi substituído.
  10. Por fim, vamos simplificar as palavras da história para atrair um público mais amplo. Podemos garantir que cada palavra na história tenha 10 caracteres ou menos usando o método .every(). O método .every() usa uma função de retorno de chamada para testar se cada elemento em uma matriz passa por uma condição especificada. Retorna true se todos os elementos passarem, e false se houver algum elemento que não passe. Para começar, defina uma variável chamada lengthCheck e configure-a para a invocação do método .every() em storyWords. Na função callback, teste se cada palavra tem menos ou igual a 10 caracteres. Exiba lengthCheck no console para ver o resultado. Se true for registrado, cada palavra na história terá 10 caracteres ou menos. Caso contrário, uma ou mais palavras terão mais de 10 caracteres.
